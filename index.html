Tech Theory


Purpose: Provide schools with a secure testing environment on Chromebooks.

When deployed in kiosk mode, this app:
- prevents the loading any additional tabs
- prevents viewing page source / answers shown in source code

Setup


Teacher setup: Create a Google Form and generate a go.gl url to use as the access code. Admin Setup: Contact your school's GSuite Administrator and have them deploy this app in kiosk mode.

Generating an Access Code


To generate an access code, visit goo.gl and generate a shortened url pointing to your Google Form.
The last 6 characters of that url (directly following the last slash after goo.gl/) is the access code.
For example, the access code for the url https://goo.gl/KJ77Gv would be KJ77Gv.
That's it.
