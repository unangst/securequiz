About


Teachers often struggle when they use try to use Google Forms as a quiz or testing platform. It seems that many students have figured out they can use the platform to 'cheat' - either viewing the answers by viewing the source code, loading a second tab or instance of Chrome, sending a link to the resource to a student elsewhere or other method(s) developed by the 'student hivemind'. The SecureQuiz App provides a secure environment for testing on Chromebooks via kiosk mode.

When deployed in kiosk mode, this app:
- prevents the loading any additional tabs or a second web browser
- prevents viewing page source / answers shown in source code
- displays a lime green bar at the top of the screen at all times & providing visual verification that Secure Quiz is Active.
- provides quick access to a Google Form (or any other web site) via an obsfucated 'access code'.

Teacher Use


Teacher setup: Create a Google Form and generate a goo.gl url shortcut to use as the access code.
(Chrome user? Check out the "goo.gl URL Shortener" in the Chrome App Store).

Generating an Access Code


To generate an access code, visit goo.gl and generate a shortened url pointing to your Google Form.
The last 6 characters of that url (directly following the last slash after goo.gl/) is the access code.
For example, the access code for the url https://goo.gl/KJ77Gv would be KJ77Gv.
That's it.

Technical Setup


Admin setup: Your school's GSuite Administrator will need to deploy this app to managed Chromebooks via kiosk mode.
This Chromebook App has been deployed to thousands of Chromebooks across multiple school districts.
Uptime - SecureQuiz.com is permanently hosted on GitHub and should have limited downtime.

Student Use


Teachers instuct students to sign out of their Chromebooks and launch the "Secure Quiz App" via the "Apps" button on the login screen. Students use the provided 'access code' to load the corresponding quiz or test. Upon completion, students can proceed to a second quiz or exit the app.
