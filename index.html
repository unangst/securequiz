About


Teachers often struggle when they try to use Google Forms as a quiz or testing platform. It seems that many students have figured out they can use the platform to 'cheat' - viewing source code to reveal answers, loading a second tab or instance of Chrome, sending the url to a student elsewhere or another method(s) developed by the 'student hivemind'. The SecureQuiz App provides a hardened testing environment on Chromebooks via kiosk mode.

When deployed in kiosk mode, this app:
- prevents loading of additional tabs or second browser
- prevents viewing page source / answers shown in source code
- displays a lime green bar at the top of the screen at all times
(providing visual verification that Secure Quiz is Active)
- supports 92 languages via a virtual keyboard. (¡Viva el español!)
- gives quick access to a quiz via an 'access code'

Teacher Use


Create a Google Form and generate a goo.gl url shortcut to use as the 'access code'.
We recommend the "goo.gl URL Shortener" for shortcut generation.
You can use the Secure Quiz app with any website or url. Have fun!

Generating an 'Access Code'


To generate an 'access code', visit goo.gl and generate a shortened url pointing to your Google Form, quiz or web site.
The last 6 characters of that url (directly following the last slash after goo.gl/) is the access code.
For example, the access code for the goo.gl shortcut https://goo.gl/KJ77Gv would be KJ77Gv. It's that simple.

Technical Setup


Your school's GSuite Administrator will need to deploy this app to managed Chromebooks via kiosk mode.
This Chromebook App has been deployed to thousands of Chromebooks across multiple school districts. SecureQuiz.com is hosted on GitHub for maximum uptime.

Student Use


Teachers instuct students to sign out of their Chromebooks and launch the "Secure Quiz App" via the "Apps" button on the bottom left hand side of the login screen. Students use the provided 'access code' to load the corresponding quiz or test. Upon completion, students can proceed to a second quiz or exit the app.
